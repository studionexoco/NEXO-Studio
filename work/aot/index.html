AOT

Character Selection Interface

Designing a cinematic character selection experience inspired by Attack on Titan’s militaristic tone and narrative intensity.

The interface explores how typography, symbolism, and tactical UI structures can convey character identity before gameplay begins.

Project Overview

This project explores a game interface system for selecting characters within a narrative-driven anime combat experience.

The design focuses on hierarchy, symbolism, and militaristic structure to create anticipation and immersion.

Role UI Designer
Platform Game Interface Concept
Domain Entertainment UI / Systems
巨

Character UI System

Each character screen follows a consistent tactical layout:

  • [01] Central character render
  • [02] Symbolic kanji background
  • [03] Name and metadata
  • [04] Descriptive narrative text
  • [05] Faction insignia

This modular system maintains consistency while expressing individual character identity.

Interaction Concept

The selection flow mimics dossier navigation through vertical character switching.

Arrows and directional indicators suggest tactical browsing between profiles, reinforcing the military theme and deliberate selection pacing.

△ ▽

UI Components

The interface combines radial rings, kanji identity shapes, grid lines, data panels, insignia placement, and navigation arrows into a unified tactical HUD visual language.

Design Reflection

This project explores how entertainment interfaces can communicate story and identity through visual structure.

By combining anime symbolism with militaristic UI systems, the design demonstrates how character selection can become an immersive narrative moment rather than a functional menu.
